系統資訊

關於 Java 虛擬機 (JVM)

使用 GraalVM 的運行效能更佳, 下載連結 點此

推薦 JVM 參數

-Dfml.ignorePatchDiscrepancies=true -Dfml.readTimeout=300 -d64 -Dsun.rmi.dgc.server.gcInterval=2147483646 -XX:+UnlockExperimentalVMOptions -XX:+DisableExplicitGC -XX:MaxGCPauseMillis=100 -XX:TargetSurvivorRatio=70 -XX:-UseGCOverheadLimit -Xnoclassgc -XX:+AggressiveOpts -XX:+CMSClassUnloadingEnabled -XX:ParallelGCThreads=4 -XX:MinHeapFreeRatio=5 -XX:MaxHeapFreeRatio=20 -XX:LargePageSizeInBytes=2m -XX:+UseCompressedOops -XX:+OptimizeStringConcat -XX:+UseBiasedLocking -XX:MaxTenuringThreshold=15 -XX:+UseConcMarkSweepGC -XX:+UseParNewGC
