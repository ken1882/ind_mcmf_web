Known issue

If you're encountering an issue, please report to my discord (@Compeador#9999)



  • ImmersiveEngineering's item silo will crash the server if there's item in its input pipe, reached the max capacity and anyone in the area left. Please consider using mekanism's bin storage. Or you'll have to block the input to the silo and extract some of its content before you leaving that chunk/logout (otherwise server will crash!).
    This has reported to sponge's issue tracker and waiting for fix.
  • Recurrent Complex's strcture will be broken inside a village
  • If Modern Wafare's guns are held in off-hand, press mouse-right button (aim-down sight) will cause the gun replaced by the item in main-hand
